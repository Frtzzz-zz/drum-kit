# Drum Kit 
Play as if you're playing real drums!

Try a simple drum fill by pressing in this sequence:
S > H > E > U > R or Y

Featuring:
- drums, that's it
- some pixel art
- can do some flams on the snare
- don't mind the tom scream, it came up first when i was searching for tom sfx, decided to include it

## Tech Stack:
- HTML
- TailwindCSS
- JS

pixel art by me (@Fritz)
SFX samples from various sources then edited and mixed on Audacity
